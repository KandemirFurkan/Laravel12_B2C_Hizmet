# Laravel12_B2C_Service

## Overview

Laravel12_B2C_Service is a B2C-focused backend application that connects users seeking services with companies providing those services.  
The platform allows users seeking services to compare quotes from different companies, while offering companies the opportunity to reach real customer requests more quickly and directly.

The goal is to create a scalable backend infrastructure that makes the service acquisition process more **transparent**, **fast**, and **efficient**.

#Core Features

- Token-based authentication  
- Role-based authorization (customer, company, admin)  
- Service request and offer management  
- Access to customer requests and offer creation for companies  
- System management via the admin panel  
- Extensible and scalable backend architecture  

---

#Target Users

- **Customer:** Creates service requests, compares quotes, and makes selections  
- **Company:** Accesses real customer requests and submits quotes  
- **Admin:** Manages the overall platform
